Hangman Game

Overview

Based on the classic children's game, you have 10 lives to guess the magic word, good luck!

    Key skills used

  • HTML
  • CSS
  • JavaScript - Using the random number functionality in JavaScript and an array of around 200 words, clicking on the refresh button will generate the appropriate amount of underscores (place holders) for the player to guess the word.
    "For loops" and arrays are utilized so that the player can guess a letter in the word. The placeholed are then filled in accordingly if the chosen letter is correct otherwise lives are decremented accordingly and the main image changed when the chosen letter is incorrect.
